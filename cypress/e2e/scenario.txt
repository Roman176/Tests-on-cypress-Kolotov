Тест на регистрацию пользователя:

Посещение страницы регистрации.
Заполнение формы регистрации данными.
Нажатие на кнопку регистрации.
Проверка успешной регистрации через проверку URL и отображения сообщения о завершении регистрации.

Тест на вход в систему:

Посещение страницы входа.
Заполнение формы входа данными.
Нажатие на кнопку входа.
Проверка успешного входа через проверку URL и отображения элементов, характерных для личного кабинета.

Тест на поиск товара:

Посещение главной страницы сайта.
Ввод запроса поиска в поле.
Нажатие на кнопку поиска.
Проверка, что результаты поиска содержат ожидаемые товары через проверку наличия элементов с классом.

Тест на добавление товара в корзину:

Посещение главной страницы сайта.
Добавление товара в корзину.
Переход в корзину.
Проверка, что корзина содержит добавленный товар через проверку наличия элементов с классом.

Тест на оформление заказа:

Посещение главной страницы сайта.
Добавление товара в корзину.
Переход в корзину.
Переход к оформлению заказа.
Заполнение данных для доставки и оплаты, нажатие кнопки подтверждения заказа.
Проверка успешного оформления заказа через проверку URL и отображения подтверждающего сообщения.

Тест на обновление личной информации пользователя:

Посещение личного кабинета пользователя.
Переход к редактированию профиля.
Изменение личной информации.
Нажатие на кнопку сохранения изменений.
Проверка успешного обновления информации через отображение подтверждающего сообщения или проверку изменений в личном кабинете.
